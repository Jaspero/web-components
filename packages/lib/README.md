# Library source (`packages/lib`)

Svelte sources for `@jaspero/web-components`. The published tarball is produced from the monorepo root with `npm run build:lib` (output: `dist/`).

Publishing to npm is handled by the **Publish** GitHub Action when `main` changes under `packages/**`, or when that workflow is run manually. See the root [README](../README.md) for release steps.
