# Link

### Description

`<jp-link>` is a component with `<link>` functionality.

### Attributes

- `disabled` (`boolean`)
- `variant` (`'basic' | 'raised' | 'stroked' | 'flat' | 'icon' | 'fab' | 'mini-fab'`) 
  - Visual style
- `color` (`'default' | 'primary' | 'accent' | 'warn'`)
- `href` (`string`)
- `target` (`string`)
  
### Slots

Used for customization of link's content.


### Methods
